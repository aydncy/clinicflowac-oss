# Roadmap (30 / 60 / 90 days)

## 0–30 days: MVP scaffold
- App shell + navigation skeleton (Flutter).
- Basic workflow: Appointment create/reschedule/cancel + reminders.
- Minimal data model + local storage (placeholder) + sample demo data.
- Basic docs: how to run, architecture overview.

## 31–60 days: Documents & consent
- Secure document upload link flow (placeholder implementation + interfaces).
- Consent capture events and consent history.
- Event log: append-only timeline for workflow steps.

## 61–90 days: Audit export + pilot readiness
- Export “proof pack” (JSON + PDF/HTML).
- Configuration for a pilot clinic (sample config, demo scenarios).
- Security hardening notes + threat model (lightweight) + privacy guidance.
