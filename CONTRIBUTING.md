# Contributing

Thanks for your interest in contributing to ClinicFlowAC (OSS)!

## Before you start
- Please open an Issue to discuss changes, especially for new features.
- Keep PRs small and focused.

## How to contribute
1. Fork the repo and create a branch: feature/<short-name> or fix/<short-name>.
2. Make your change.
3. Open a Pull Request with:
   - What changed
   - Why it changed
   - How to test

## Reporting security issues
Please do not open public issues for security vulnerabilities.
Use the contact details in SECURITY.md.
