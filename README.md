# ClinicFlowAC

Reference client for **Open Verifiable Workflow Infrastructure (OVWI)**.

ClinicFlowAC demonstrates how informal operational communication (WhatsApp, Email, SMS, phone) can be transformed into structured, verifiable, and audit-ready workflows.

This repository is not merely a clinic tool.

It is a reference implementation of open execution infrastructure designed for regulated environments.

---

## Part of the OVWI Ecosystem

ClinicFlowAC works together with:

- **OVWI (Open Verifiable Workflow Infrastructure)**  
  Open audit-ready workflow engine for append-only, verifiable workflows.  
  https://github.com/aydncy/ovwi-oss

Healthcare is the demonstration domain.

The infrastructure is sector-agnostic and designed to be reusable across healthcare, finance, and other regulated environments.
